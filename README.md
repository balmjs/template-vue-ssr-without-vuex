# BalmJS template - `vue-ssr-without-vuex`

> A server-rendered Vue project without Vuex

## Requirements

```sh
npm install -g balm-cli balm-core
# OR
yarn global add balm-cli balm-core
```

```sh
# Verify
balm -V

# Output:
# balm-cli: 4.0.0
# balm-core: 4.0.0
```

## Installation

```sh
balm init vue-ssr-without-vuex my-project

cd my-project
yarn
# or
npm i
```

> (Recommended) [BalmUI](https://material.balmjs.com/) for your project

## Enjoy

```sh
# For development
npm run dev
node server.js # run dev server for test api

# For production
npm run prod

# For running server
npm run start
```
